# **Project(prog2)**
## Nyelvtanuló app

_A japán írás kanji karaktereinek megtanulásában segítő app._

* ### Nyelv:

    _Java_

* ### Platformok:
  
    _Android_




## Az appról: 
---
* ### Elsődleges elemek

    - kanji/jelentés kiválasztása az adatbázisból
    - szó bevitele

    - egyezés ellenörzése

* ### Másodlagos elemek

    - spaced repetition system használata
    - kanji csoportosítása nehézségi szint szerint
